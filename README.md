# Data Jobs Dashboard with Power BI

This project is a **Power BI dashboard** built for **Job Seekers, Job Transitioners, and Job Swappers** who want a clear, data‑driven view of the data science / data job market.

Using a real‑world dataset of **2024 data science job postings** (including job titles, salaries, and locations), this dashboard transforms scattered information into a single, intuitive interface for exploring market trends, compensation, and skills demand.

---

## 📊 What this dashboard does

- **Unifies scattered information** about data jobs into one interactive report.  
- Lets you explore **trends, median salaries, job counts, and skills** by title, location, and work‑from‑home options.  
- Helps you make **informed decisions** about which roles and skills to target in your next career move.

---

## 🚀 Dashboard versions

This repository contains two main iterations of the project:

### 1. Data Jobs Dashboard 1.0

- Original **multi‑page Power BI dashboard** with:
  - A high‑level market view (KPIs, top titles, and salaries).  
  - A detailed **Job Title Drill‑Through** page for deep‑dive analysis (salary ranges, work‑from‑home stats, hiring platforms, and global job locations).  
  - Interactive features: slicers, buttons & bookmarks, drill‑through navigation, and multiple chart types.

> This version focuses on **exploring the job market in depth** with a two‑page layout.

### 2. Data Jobs Dashboard 2.0 (Single‑Page)

- A **refined, single‑page dashboard** built from lessons in V1.  
- Designed to give job seekers the **most important insights at a glance**, including:
  - Job Count, Skills Per Job, Median Yearly & Hourly Salary.  
  - Skill Popularity (by job percent or count).  
  - Salary comparisons across Job Titles.  
- Uses **Power Query ETL, DAX calculations, and interactive features** (slicers, buttons & bookmarks) on one focused canvas.

> This version prioritizes **quick, efficient analysis** instead of multi‑page navigation.

---

## 🧰 Key skills showcased

Across both versions, this project demonstrates:

- **⚙️ Data Transformation (Power Query ETL)**  
  Cleaning, reshaping, handling blanks, changing data types, and creating new columns.
- **🔗 Data Modeling (Star Schema)**  
  Building clean relationships and a solid data model.
- **🧮 DAX & Measures**  
  Creating KPIs like Median Yearly/Hourly Salary, Job Count, and Skill Popularity.
- **📊 Visualizations**  
  - Column, Bar, Line, Area charts for trends and comparisons.  
  - Map charts for global job distribution.  
  - Cards for KPIs and Tables for granular detail.
- **🎨 Dashboard Design**  
  Crafting intuitive layouts and selecting chart types that tell the data story clearly.
- **🖱️ Interactivity**  
  Slicers, buttons & bookmarks, drill‑through navigation, and responsive filters.
